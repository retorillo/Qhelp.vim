# Qhelp.vim

[![MIT](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Vim plugin to quickly glance help section

![preview](preview.gif)

## Install (Pathogen)

```vimL
git clone https://github.com/retorillo/qhelp.vim.git ~/.vim/bundle/qhelp.vim
```

## Usage

```vimL
:Qhelp {helptag}
```
